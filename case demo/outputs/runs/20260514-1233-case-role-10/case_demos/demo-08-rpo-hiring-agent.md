# 招聘辅助Agent：RPO企业的人岗匹配提效

- case id: CASE-008
- 行业：专业服务
- 能力：组织与人才
- 公开客户画像：区域RPO与人才服务机构

## Case Demo

### 背景
人才服务机构本来就处理大量简历、岗位和匹配动作，一旦人效触顶，业务增长就会被流程卡住。

### 挑战
简历初筛效率低、岗位匹配不稳定、交付节奏靠人工经验，难以持续放大服务能力。

### 需要的外部能力
需要懂HR业务、产品思维和AI工具的人，把匹配逻辑做成可复制的系统。

### 介入方式
先把岗位要求、简历筛选和人才匹配做成标准流程，再把辅助Agent嵌进去提升筛选和沟通效率。

### 结果方向
人岗匹配更快，服务团队能把更多时间放在高价值判断和客户沟通上。

## Role Demo

### 推荐人才画像
AI-HR产品负责人 / 人才匹配专家。先看HR业务理解，再看AI和产品化能力。

### 关键能力
- HR流程理解
- 人岗匹配逻辑
- 产品化思维
- 数据驱动优化
- 跨部门协同

### 90 天工作重点
1. 梳理筛选和匹配流程。
2. 定义岗位和简历的标准标签。
3. 先做初筛和推荐两个节点。
4. 复盘匹配质量和交付效率。
