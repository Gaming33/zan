# 直播脚本流水线：服饰品牌的AIGC内容工厂

- case id: CASE-004
- 行业：消费与零售
- 能力：增长与市场
- 公开客户画像：全国连锁服饰品牌

## Case Demo

### 背景
服饰品牌的内容消耗很高，短视频、直播和活动素材都需要持续供应，人工创作跟不上业务节奏。

### 挑战
高频直播吃掉大量内容产能，脚本、分镜和投放素材都高度依赖人，增长链路不稳定。

### 需要的外部能力
需要能把内容生产、投放和转化串起来的人，既懂品牌增长，也懂AIGC工作流。

### 介入方式
把脚本规划、分镜、素材生成、直播辅助和发布节奏连成流水线，让内容生产从单点创作变成规模化流程。

### 结果方向
内容产能更稳，脚本生成和投放节奏更快，品牌可以把增长动作做成标准化流程。

## Role Demo

### 推荐人才画像
AI增长负责人 / AIGC内容运营负责人。重点看增长逻辑、内容组织能力和AI工作流设计能力。

### 关键能力
- 内容增长策略
- AIGC工作流设计
- 直播和短视频运营
- 选题、脚本和分发协同
- 指标复盘能力

### 90 天工作重点
1. 识别内容生产最耗时的 3 个节点。
2. 把脚本和素材流程标准化。
3. 搭建试点增长闭环。
4. 复盘内容产能和转化变化。
