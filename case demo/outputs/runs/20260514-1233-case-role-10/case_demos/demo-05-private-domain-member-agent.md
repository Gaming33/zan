# 私域会员Agent：母婴品牌的推荐与导购重构

- case id: CASE-005
- 行业：消费与零售
- 能力：增长与市场
- 公开客户画像：全国连锁母婴零售品牌

## Case Demo

### 背景
母婴零售很依赖复购和会员运营，但公域获客越来越贵，私域触达和导购效率却不够稳定。

### 挑战
会员推荐粗放、私域运营分散、导购人效不高，导致复购和转化都难持续放大。

### 需要的外部能力
需要能把会员运营、内容触达和导购动作做成一个完整Agent工作流的人。

### 介入方式
把标签、触达、推荐和导购动作连成闭环，先从高频会员场景做自动化，再逐步扩展到更复杂的私域运营动作。

### 结果方向
私域触达更精准，导购动作更统一，会员运营逐步从人工经验转向系统驱动。

## Role Demo

### 推荐人才画像
AI私域增长专家 / 会员运营负责人。优先看消费业务理解，再看AI和自动化能力。

### 关键能力
- 私域和会员运营
- 推荐和触达逻辑
- 导购提效
- 内容与转化协同
- 数据复盘

### 90 天工作重点
1. 梳理会员运营和导购的关键流程。
2. 找出最适合先自动化的触点。
3. 建立推荐和触达试点。
4. 做成可复制的私域动作模板。
